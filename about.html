Bookshelf


About The Project:


BOOKSHELF is a Software Engineering Project assignment done by students of Maulana Azad National Institute of Technology, Bhopal under the guidance of Prof. Narasimha Naik Banavath. This is a prototype of what the actual website must have looked like. It consists of all the active links but doesn't contain the required data as the database needs to be updated as per the requirements of asking authority.

BOOKSHELF is an online store where all possible facilities of book trade, that is buying, selling, taking and giving on rent is made possible. It (not in the prototype) provides a variety of books of numerous different fields which can be taken or given on rent if not buy them. BOOKSHELF also provides various e-books which can be easily downloaded once you have created an account with us.
One of the most prominent features of BOOKSHELF is that it also provides with the hand-written notes of students from the elite institutes of the world.

BOOKSHELF is a one-stop point where one never returns disappointed.
